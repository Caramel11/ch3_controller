# 后半段高刚度区域波动分析与优化记录

日期: 2026-06-24

## 问题现象

最近几次 Gazebo 可视化实验中，扫描后半段进入高刚度区域后，力跟踪误差与位置跟踪误差出现更明显的周期性波动。该问题主要发生在 Phase 3 正式扫描阶段的高刚度尾段，而接近、扫描前段和复位阶段整体稳定。

## 原因分析

对基线实验 `no_rcm_20260624_045051` 的高刚度区进行相关性分析后，后半段力误差并不是主要由 alpha 本身突变引起。高刚度区内，力误差与 z 方向微小压入误差强相关，环境刚度约为 500 N/m 时，0.05 mm 级 z 方向波动即可带来约 0.025 N 的力波动。

关键判断:

- `F_err` 与 `pos_z_err` / `delta` 的相关性最高，说明 z 向压入深度的微小周期误差是主要来源。
- alpha 在高刚度区变化较平滑，直接降低 alpha 会降低平均作用力，但可能放大位置跟踪滞后。
- x 方向存在约 2.2 mm 的稳定滞后；当扫描速度不变而环境刚度升高时，该滞后通过接触模型和 z 向闭环耦合放大后半段波动。
- 高刚度尾段更需要降低激励频率，而不是单纯提高刚度或激进调小 alpha。

## 本轮保留的修改

在 `run_no_rcm.py` 中新增基于估计刚度的扫描速度调节:

- 低刚度和中刚度区域保持原扫描速度 `scan_vx = 0.0016 m/s`。
- 当 `K_hat` 从 420 N/m 过渡到 500 N/m 时，用 smoothstep 平滑降低扫描速度。
- 高刚度区最小速度比例为 `0.75`，避免速度突变造成新的激励。

该策略保留 alpha 调节律主体不变，只降低高刚度区的轨迹激励强度，因此风险比重新整定博弈控制器 `Q/R` 或大幅改变 alpha 规则更小。

## 后续可选优化

1. 高刚度区可继续尝试 `scan_vx_stiffness_min_scale = 0.65-0.70`，预期可进一步降低力波动，但扫描时长会继续增加。
2. 若优先缩短实验时间，可保留当前高刚度降速，同时提高低刚度区 `scan_vx`，避免全程变慢。
3. 若后续继续调博弈控制器 `Q/R`，建议只针对 z 向力误差与 z 向位移误差增加阻尼权重，避免影响 x/y 轨迹跟踪和姿态稳定。
4. 不建议为了改善指标而直接改变虚拟环境刚度分区；那会降低实验挑战性，不能真实说明控制器在刚度突变区域的适应能力。
